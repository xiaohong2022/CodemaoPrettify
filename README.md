# CodemaoPrettify
美化编程猫主页的一个小插件

## 目前支持：
1.更改主题色

2.更改排版（导航栏、首页、发现、论坛、活动）

3.使用涟漪（采用MDUI）

4.帖子防吞格式（要安装了该插件才可以防）

### 更多敬请期待

## 如何安装？

1.先安装一个脚本管理器（例如：油猴 `tempermonkey`、脚本猫 `ScriptCat`）

2.点击“新建脚本”

3.点击保存 `Ctrl+S`

4.成功安装，打开[编程猫首页](https://shequ.codemao.cn)即可预览
 
## 如何使用？

1.打开[编程猫首页](https://shequ.codemao.cn)

2.点击导航栏的“…”

3.点击“CodemaoPrettify”打开进行设置

4.设置完成后点击保存
